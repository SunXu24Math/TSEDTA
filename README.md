# TSEDTA
## Archival DOI
This repository contains the official implementation for the paper: **"TSEDTA: A Transformer-based neural network with ESM2 and SMILES Transformer embeddings for drug-target binding affinity prediction"**.

To ensure long-term reproducibility, a persistent snapshot of the code and datasets used in the manuscript is archived on Zenodo:

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19103249.svg)](https://doi.org/10.5281/zenodo.19103249)

> **Note:** For the latest updates and bug fixes, please refer to this GitHub repository. For citing or reproducing the exact results from the paper, please use the archived version on Zenodo.
